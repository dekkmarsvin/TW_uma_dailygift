# 專案目標

## 1. 核心功能

自動開啟指定的網站(https://uma.komoejoy.com/event/dailygift/)，並自動點擊**簽到**![alt text](image.png)領取獎勵。

## 2. 技術目標

- 實現自動登入功能。
- 實現自動領取獎勵功能。
- 實現cookie的自動儲存與讀取，讓使用者不需要每次都登入。
- 實現cookie失效時的自動重新登入。
- 紀錄領取獎勵的成功與否，紀錄於log檔案中。

## 2.1 執行流程

1. 自動開啟指定的網站(https://uma.komoejoy.com/event/dailygift/)
2. 自動點擊**登入**按鈕
3. 自動輸入帳號密碼
4. 自動勾選同意條款
5. 自動點擊**登入**按鈕(登入按鈕需要勾選同意條款後才能點擊)
6. 自動點擊**簽到**按鈕
7. 紀錄領取獎勵的成功與否，紀錄於log檔案中。
8. 關閉瀏覽器。

## 3. 額外功能

![alt text](image-1.png)

總積分(本年度積分+即將過期積分)>=100時，點選抽獎(class="points-draw")

流程

1. 檢查本年度積分與即將過期積分
2. 檢查是否>=100
3. 如果是，檢查獎勵points-show-box-name是否還有庫存(剩餘>0)，點選抽獎
4. 紀錄抽獎結果
